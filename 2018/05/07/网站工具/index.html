网站工具

发表于 2018-05-07   |  阅读量 次 | 分类于 程序员网站 | |
阅读时长 ≈ 3

今天给大家整理了几个非常实用的网站工具，也是我一直都在用的，所以来分享给大家，工具的介绍以及使用方法在下面会有介绍，大家可以按需学习使用。

1、看雪学院 ：信息安全黄埔军校

一个集安全编程、软件安全测试、智能设备安全、物联网安全、漏洞挖掘、移动安全、WEB安全、密码学、逆向技术、加密与解密与系统安全等全一身视频课程网站，选择它你一定没错。

2、吾爱破解 ：技术分享论坛

这是我常逛的一个网站，里面有各种技术分享，包括很多教程与软件的破解版，也包括最新的发现，比如QQ坦白说、cf辅助破解、unity3d手游以及小程序源码等好东西，反正当自己有需求时，不妨进此网站逛逛也是一个不错的选择。

3、埃文+ ：IP高精度定位

不得不多，这玩意真不错，IP定义地图，可以说是真活地图。

4、 Leetcode ： 在线编程

如果说选择在线编程，就进Leetcode，在这里你可以从一个小白进阶到的中级程序员高度。此外我介绍几个编程学习网站，像鱼C工作室、52CTO、菜鸟教程等，希望你能在这里学有所长。

5、云盘精灵 ：万能网盘

在这里共收录了87万数据网盘资源，每天都在更新，如果当你想要找资源，却发现搜索引擎里找不到，那么来这里吧，一定能找到你满意的资源，供你下载。以此我再介绍一个它的同袍兄弟《胖次搜索》，这里可能更全，支持很多类型，包括影音、歌曲、图片壁纸、小说、程序App、BT种子以及压缩文件。

6、ProcessOn ：免费在线作图

ProcessOn是一个免费的在线作图网站，可以在线制作流程图、思维导图、UI原型图、UML、网络拓扑图、组织结构图，制作完后可以直接保存在云端，就不用担心更换设备的问题。此外网站里提供了大量的制图模板，只要自己稍作修改，很快就可以制作出属于自己的流程图。另外还可以支持团队在线协作，直接将作品共享给好友。

7、Kindle ：电子书阅读器

下载kindle PC版本，打开之后，在主页工具找到选项，点击内容可以找到本地内容的文件位置，一开始默认在C盘，可更改，把azw3文件放进去即可阅读。补充：kindle PC在电脑上的阅读体验一般。下面是一本书文件名.azw在kindle上的效果.
