教你如何使用ShadowsocksR学会翻墙

发表于 2018-04-23   |  阅读量 次 | 分类于 翻墙方法 | |
阅读时长 ≈ 2

这里我给出一个链接，特别棒免费SSR帐号分享 ，那么接下来又该如何操作。首先我们前提是要下载安装SSR，云盘下载链接ssr4windows ，打开上面的账号分享链接，界面如下，这里的SSR翻墙账号每天都有在更新，但我所知道的，其中必有几个是长期都可以使用的，鼠标移到V2RAY节点，右击复制链接。

最后成功翻墙进入谷歌界面如下。如有同学特别想翻墙，此文会对你受益匪浅。同时电脑与手机是可以同步翻墙的，我现在基本上不用百度了，能用谷歌干嘛要用百度呢？

1、双击打开ShadowsocksR-win.7z 文件，然后把ShadowsocksR-win.exe 文件解压到电脑桌面。

注：Win7用户请使用ShadowsocksR-dotnet2.0.exe 文件；

Win8或Win10用户请使用ShadowsocksR-dotnet4.0.exe 文件。

2、输入本站产品服务提供的 节点地址、连接端口、加密方式选择 aes-256-cfb、连接密码，点击确定，如图：

这里我主要是介绍在你没有账号和密码的情况下又如何免费使用翻墙（你要是有的话，不花钱买个国外的服务器，这可是要花一笔不菲的资金 ）所以你需要打开上面的免费SSR账号分享链接，界面如下，这里的SSR翻墙账号每天都有更新，但我所知道的，其中必有几个是长期都有用的，鼠标移到V2RAY节点，右击复制链接。

接下来就是将复制的链接直接导入，在服务器选择刚刚你复制链接的端口。

3、选择 系统代理模式，点击全局模式。

4、选择代理规则，点击全局模式。

5、最后，打开浏览器，访问Google搜索或者Youtube，如果可以访问，证明已经可以正常使用。这是我用谷歌打开的界面，完全可以使用谷歌、facebook与Youtube.
