无扩展，不谷歌

发表于 2018-05-03   |  阅读量 次 | 分类于 谷歌干货 | |
字数统计: 2k | 阅读时长 ≈ 6

相信现在很多人都在使用谷歌浏览器，其流畅的浏览体验得到了不少用户的偏爱，但流畅只是一方面， Chrome 最大的优势还是其支持众多强大好用的扩展程（Extensions）。今天我来推荐了一些 Chrome扩展，把10款优质的Chrome扩展集中推荐给大家。谷歌插件是个贼好玩的东西，用了这些插件之后，你不得感慨谷歌真是强大到了令人窒息。但你要想添加这些扩展，需要在谷歌应用商店下载，必须翻墙，所以你得有足够的条件。在前文我已经介绍了一种翻墙方法，其实还有很多，比如自由门7.58、无界浏览15.04以及动网通3.2均可以翻墙进入谷歌，包括你自己搭建ssr账号。关于翻墙的原理，其实相当简单的，送你三个词，找度娘，这里我就不强调了。

首先我来介绍了一些绝杀的谷歌插件，“无它，不无谷歌”也是它的代言，用过谷歌的人都知道，这些插件强大到了无敌，作为互联网的新星人才，这些插件都是必备的。

一、微博图层：提供图床服务

我们都知道，当你有了个人的网站后，会发现你存储图片与视频的容量是相当低的，如果我们要放的图片与视频过多，那是不太够用，所以这时候我介绍一款谷歌插件，那就是微博图床。安装成功后，会在我的谷歌浏览器右上角会出现这样的图标。如果你没有这个翻墙的条件，这些插件可以去镜像站点安装。

二、分享 ：强大的网址

在这里你可以看到很多你想要的地址，回想起来我们平时阅读的一些东西太没有营养价值了，都是一些小儿科的东西 ，而在这里你可以看到你想不到的黑幕。首先我是这个插件的受益者，从这个插件我学过了自己所涉及不到的领域知识，让我看到了还有这么强大的东西。相信你们也是同样可以受益于其中。

三、Anything to QRcode ：识别 / 生成二维码

现在很多网站都会提供二维码入口，方便用户通过手机扫描 / 识别的形式快速访问，但对于电脑客户端而言其实并不方便。比如说有些应用下载不提供超链接，而仅提供二维码；有时候需要通过手机浏览器访问电脑上当前访问网页，但如何将链接发送到手机也够让人头疼。

这款扩展就是用来解决这两个问题的。

第一、没设置页面但用起来也非常简单：如果你希望将当前网页生成二维码，那么在工具栏中找到 Anything to QRcode 图标点一下，就会立即生成方便访问；

第二、 Anything to QRcode 还可以解析二维码，比如在二维码上点击鼠标右键，在弹出的右键菜单中选择「Anything to QRcode - 解析二维码」，解析后的结果或者链接就会出现在页面上方方便复制。

四、LastPass ：免费的密码管理平台

就浏览器密码保存这项服务，我选择了LastPass，一个是相对配置很简单 ，在默认表单的基础上可以几乎不做太多改动就可以通用，第二个是方便的密码填选功能 ，可以针对性在某一个站点存储多个密码。毕竟在目前的网络环境中，指望Chrome中Google账户的保存密码是不现实的。所以 LastPass 只能是我其中备选项之一。但是我这里需要说明的是 LastPass 跨平台使用是需要另外付费的。

五、Advanced Font Settings ：高级字体设置

我深知 Windows 平台和 macOS 平台的差异，尤其是很多 Geek 老生常谈的「字体渲染」问题，这种变化放在 Chrome 这种跨平台应用上依旧存在。同样是默认设置下，macOS 平台的 Chrome 字体看上去就是比 Windows 好看。虽然可以通过样式、修改注册表更换渲染字体的方法解决问题，但要知道折腾起来没点技术基础还真搞不定。

具体到 Advanced Font Settings 中就是：

  • default：默认西文；
  • standard：使用 Noto Sans；
  • serif：使用 Noto Serif；
  • Sans-Serif：使用 Noto Sans。

对应到中文，在 script 中选到 Simplified Han，然后：

  • Standard：使用 Noto Sans CJK SC Regular；
  • Sans-Serif：使用 Noto Serif CJK SemiBold；
  • Sans-Serif：使用 Noto Sans CJK SC Regular；
  • 下方对应的 Fixed 都和 standard 保持一致。

设置完成重启 Chrome，再打开网页，是不是觉得字体变得好看很多？

六、化词翻译 ：强大的多引擎划词翻译工具

虽说 Chrome 浏览器内置有 Google 自家的翻译全文至 xx 语言的工具，但这个工具并不能逐句翻译，显得并不是那么「人性化」。而这款名为「划词翻译」的 Chrome 扩展除了可以对选中段落进行翻译之外，还可以在翻译时选择多个翻译引擎，除了我们熟知的「Google 翻译（分为 Google 翻译国内和国外两种引擎）」，还支持有道、百度以及必应引擎，弹出的翻译结果迷你框还可以固定在网页中与原文进行对照，非常方便。

七、股票小助手 ：办公室的炒股神器

作为我的专业课，炒股已经是一件常事，但如何可以方便看股，这时候一个简单的谷歌插件就解决这个问题。在最近一年股票市场异常火热，但要想在工作上网时候都会不经意地瞄一下大盘和自己的自选股，这个名为股票小助手的 Chrome 扩展应该最为合适：只需要点击扩展栏中股票小助手图标，就可以在弹出的迷你窗口看到当前大盘的涨跌情况、最新价、涨跌幅以及换手率，在搜索框中输入代码或者股票拼音还可以自动加入你的自选股中，鼠标移动到个股上还可以显示该个股的走势曲线。不过这个扩展也只能看盘，交易还是需要用手机或者交易软件搞定。

八、Lucidchart 离线图表 ：在线绘制多种图表

这个扩展程序是一款多功能绘制程序，支持绘制流程图，思维导图，版面设计等，并且有在线和离线两种模式，可谓功能强大。

九、Pixlr Editor：试试这款「在线版」Photoshop

Pixlr Editor 是由专业图像应用服务商 Autodesk 开发的在线图片编辑器，界面几乎就是 PhotoShop 的翻版，支持 PhotoShop 的图层以及 PSD 文件导入，而我们常用的套索工具、选择工具、魔棒工具也都统统支持，Photoshop 操作的朋友应该能很快上手，比较让人感动的是 Pixlr Editor 无需注册就可以免费使用。

十、节省流量程序 ：为你省下一些流量

Google 官方推出的省流量扩展程序，启用之后 Chrome 会自动压缩被访问网页的数据来节省流量，但不包括 SSL 加密协议的网页，这个版本还处在测试阶段。另外，iOS 和 Android 版本的 Chrome 可以在设置中启用。

以上就是我目前正在使用的Chrome扩展，相信同学们看完之后一定能找到自己青睐的扩展，提高生产力就从现在开始吧。
