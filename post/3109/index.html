为什么要写博客？

发表于 2018-04-13   |  阅读量 次 | 分类于 生活杂谈 | |
字数统计: 1.6k | 阅读时长 ≈ 5

一、关于写博客

为什么要写博客呢，其实思考很久了，之前有想写博客的冲动。这次通过hexo与github搭建我的个人博客，而个人博客可以拥有更高的自由度，它有很多漂亮的主题。我看到到很多非常好的博文之后，在我迷茫未来，苦苦寻求的时候，读到了他们这些文字，让我看到了方向，于是有了我现在的博客，我也要通过努力来实现自己的梦想，不再日复一日地去浪费时间，希望以后我能一直坚持写下去。

二、写博客的意义

  1. 记录学习 。我们都知道，学习新知识之后，在短时间内尽快复习，会加深自己对知识的记忆，所以每当我们学习一些知识之后，尽快地去复习，那么有一个自己的博客是个很好的事情，来到自己的博客，将自己所学到的知识结合自己的理解记录下来，并且在以后忘记了也可以翻出来看看。那么博客在这里就提供了一个复习、记录知识的平台。

  2. 思考和学习的过程。当我们将自己学习的知识在往大脑输入的时候，我们需要思考，同样我们在输出的时候也需要思考。 你所想表达的东西在你大脑里可能是线性的，就像一维数组那样。但也可能是一棵树，或许是一棵二叉树 ，也可能会是一张图。但你将你所想说的用文字来实现的时候，就需要把它们都转成线性的。这个过程就好像一种叫做“堆”的数据结构，他的逻辑结构是一棵树，但却是用物线性的数组来存储。在这个转成线性的过程中，我们需要来分析这之间的逻辑结构，思考先说哪一部分，然后再说哪一部分，这样输出的过程就是一个用知识的过程，知识记到脑袋里并不一定是自己的，要自己能说出来，能应用才算真正的学会。所以写博客这个过程其实就是一个思考的过程，也是一个学习的过程。

  3. 检测你最近是否在学习 。当你不知道写什么的时候，想想自己最近是不是没有学习和读书。学计算机专业的同学都知道程序要给一个输入，然后经过计算机之后才有相应的输出。读书的过程就像一个输入的过程，写作是一个输出的过程。你发现你没有输出的时候，自然应该考虑是不是没有输入。所以当你不知道该写什么的时候，不是你拖延，而是因为你最近可能没有接受什么新的知识。 还有相似的例子，那就是你跟别人聊天时，你所提供的谈资，假如说你在宿舍睡大觉，睡了好多天，然后你出去跟别人聊天，你并不知道外界发生了什么，有什么好玩的事情，那自然不知道该说些什么。

  4. 展示你的能力 。这一点的前提是你自己的博客是一个技术博客，或有一大部分是技术笔记。这正是我写博客的很重要的一点，就是为了记录自己学到的新知识。 当你要接受面试的时候，我们当然知道公司面试的目的是为了选出更加优秀的人才，但是仅仅靠做那一份题，几个小时的面试是很难展现出一个人的真实水平。但是博客却不同，博客应当记录你几个月，几年的写作情况，这个是你最真实的展现。一看你的博客，就知道你在哪些领域特别熟悉，你都是学过什么技术。所以对于以后面试来说，博客是加分项。

  5. 与世界分享你的知识，经验和见解 。一个好的博客会让你交到志同道合的朋友。传统的交朋友的方式都是线下的，两个人需要不断地谈话才能相互了解，这样一个过程是比较缓慢的。而博客就像是一本人物传记一样，我们在博客中的写作会很容易展现的我们的态度，经历，价值观。所以如果一个人认真地看了这些，并且认同你的观点，一定会有想认识你的冲动。那么博客会起到展示你自己的作用，会给你带来一些志同道合的朋友。

三、关于本博客

在博客中记录自己的生活，在回头看的时候，不至于什么都想不起来，记录生活可以让我们更好地审视过去，更好地思考未来。在博客中写技术笔记，会不断地增加我们的知识和能力。在博客中写一些自己喜欢的东西，会展示出我们的其他方面的爱好和特长。

看完这篇博文不知道你有没有想法弄一个自己的博客，其实不是很麻烦的。 我这个博客是用hexo+github搭建的，完全免费，但要自己花点钱买了一个域名。主要参考了马攀同学的教程 零基础免费搭建个人博客网站 如何搭建个人独立博客？ 认识与入门 Markdown ，如果你像我一样是个小白，一个教程可能不够，某一个坑进去就不知道咋爬出来，多看两个，才会在细节上尽量地不出错。
